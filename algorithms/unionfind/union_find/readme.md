# 基本的なUnion-Find

## 概要
Union-Find（またはDisjoint Set Union）は、データ構造の一つで、以下の操作を効率的に行うことができます。

- **Find（根の探索）**:
  要素が属する集合の代表（根）を見つける。
- **Union（集合の併合）**:
  2つの集合を1つにまとめる。
